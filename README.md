# Amazon Reviews Sentiment Analysis

## 📌 Project Overview
This project performs sentiment analysis on Amazon product reviews using Natural Language Processing (NLP). It classifies reviews into Positive, Negative, and Neutral sentiments and generates insights from customer feedback.

## 🎯 Objectives
- Analyze customer opinions from product reviews
- Classify sentiment using TextBlob
- Visualize sentiment distribution
- Extract meaningful business insights

## 🛠️ Technologies Used
- Python
- Pandas
- TextBlob
- Matplotlib

## 📊 Features
- Data Cleaning & Preprocessing
- Sentiment Classification (Positive, Negative, Neutral)
- Data Visualization (Bar Chart, Pie Chart, Custom Chart)
- Insight Generation

## 📁 Dataset
- Amazon Fine Food Reviews Dataset
- First 5000 rows used for analysis

## 📈 Results & Insights
- Majority of reviews are positive
- Negative reviews mainly highlight product quality and delivery issues
- Neutral reviews are minimal compared to positive/negative
